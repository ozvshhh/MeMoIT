## 🧭 Navigation (네비바)

메모잇의 핵심 네비게이션은 아래 4개 탭으로 구성됩니다.

- **홈**: 분석 완료된 영상 목록 & 추천 태그 노출  
- **추가**: URL 추가 → AI 분석 프로세스로 이동  
- **카테고리**: 자동 분류된 키워드 기반 카테고리 탐색  
- **저장**: 사용자가 북마크한 메모 모음

---

## 🏠 Home (홈)

- 메인 화면에서는 AI가 분석한 영상들의  
  **추천 태그, 썸네일, 제목**을 한 번에 볼 수 있습니다.
- 영상(메모)을 클릭하면 아래 정보들을 확인할 수 있습니다:
  - AI 요약 내용  
  - 사용자가 정리한 개인 메모  
  - 원본 링크  
- 검색/필터 기능을 통해 **카테고리별 메모 탐색**도 가능합니다.
